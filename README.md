# Photo app.

## App omschrijving:

Met deze app kan je gebruikers accounts aanmaken en als je eenmaal ingelogged ben kunnen er pas foto's worden gemaakt.
Ook kan je jou huidige locatie zien (alleen als je bent ingelogd).

## Packages: 

* meteor-platform
* mdg:camera
* accounts-ui
* accounts-password
* mdg:geolocation
* autopublish
* insecure
